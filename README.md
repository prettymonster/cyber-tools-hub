# 💻 Cybersecurity Tools Hub

A simple website that provides tutorials and resources for popular cybersecurity tools.

## 🌐 Live Website
👉 https://prettymonster.github.io/Cyber-Tools-Hub/

## 📌 Features
- Dark hacker-themed UI
- Beginner-friendly design
- Direct links to tool tutorials
- Easy navigation

## 🛠️ Tools Covered
- Kali Linux
- Wireshark
- Nmap
- Burp Suite
- Bettercap
- Maltego

## 📂 Project Structure
cyber-tools-hub/ │── index.html │── tutorial.html │── images/
## 🚀 How to Use
1. Open the website  
2. Click on any tool  
3. Get redirected to tutorial
